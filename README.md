# GDPR Website Compliance Checker
This project is a lightweight GDPR Website Compliance Checker for Irish SMEs.
## Features
- HTTPS / SSL check
- Cookie header check
- Privacy policy detection
- Tracker indicator detection
- Form structure analysis
- AI-assisted form review
- Weighted compliance score
- Streamlit interface

## Files
- `app.py` – Streamlit app
- `scanner.py` – compliance checks
- `ai_assessor.py` – AI form analysis
- `requirements.txt` – dependencies
